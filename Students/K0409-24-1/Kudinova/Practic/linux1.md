Лабораторная 1:
Использована команда: sudo awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head -10. 
awk '{print $1}' извлекает IP-адрес из первого столбца лога. 
Sort сортирует IP для uniq. 
Uniq -c подсчитывает количество повторений каждого IP. 
Sort -rn сортирует по убыванию числа запросов. 
Head -10 выводит первые 10 строк. 

Для JSON-лога access_json.log использована команда: sudo jq -r '.remote_addr' /var/log/nginx/access_json.log | sort | uniq -c | sort -rn | head -10. Jq -r '.
remote_addr' извлекает значение поля remote_addr из JSON без кавычек. 
Далее sort, uniq -c, sort -rn, head -10 работают аналогично. 

Логи удалены командой: sudo rm -f /var/log/nginx/access.log /var/log/nginx/access_json.log.
