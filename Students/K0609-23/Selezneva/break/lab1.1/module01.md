# Модуль 1 — процессы, демоны & systemd

## Задача
Изучить управление процессами и написать systemd unit для скрипта, который должен автоматически перезапускаться при падении.

## Решение
Я создала файл сервиса в `/etc/systemd/system/`. В конфигурации использовала параметр `restart=always`, чтобы обеспечить автоматический подъем процесса. Работу сервиса проверяла через статус в `systemctl`.
