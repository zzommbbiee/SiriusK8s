# 1. Nginx access.log - топ‑10 IP + освоение vim и mc

Генерируется большой лог (1 млн строк), с помощью конвейера `awk`, `sort`, `uniq` выводится топ‑10 IP‑адресов по числу запросов. Дополнительно разбираются JSON‑логи через `jq`.

В процессе работы осваиваются базовые приёмы **vim** (режимы, навигация, редактирование, поиск, замена) и **Midnight Commander** (навигация по панелям, операции с файлами через F‑клавиши, просмотр и редактирование файлов). Таким образом, навыки анализа логов закрепляются практикой работы с двумя ключевыми консольными редакторами.
