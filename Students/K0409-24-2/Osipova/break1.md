В первой лабораторной работе требовалось найти 10 IP-адресов с наибольшим количеством запросов в логе nginx (/var/log/nginx/access.log, 1M строк).
Для решения: awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head -10
Команда awk '{print $1}' извлекла первое поле из каждой строки лога — IP-адрес клиента. Команда sort упорядочила IP для последующего подсчёта. Команда uniq -c посчитала количество повторений каждого IP. Команда sort -rn отсортировала результаты по убыванию количества (числовой сортировкой). Команда head -10 вывела топ-10 записей.
Дополнительно: jq -r '.remote_addr' /var/log/nginx/access_json.log | sort | uniq -c | sort -rn | head -10
Утилита jq распарсила JSON и извлекла поле remote_addr, остальной конвейер аналогичен работе с текстовым логом.
