# <b>ЧЁ СДЕЛАЛ?</b>
Скрипт сгенерил лог файл в 1 млн строк. Для топ-10 адресов по частоте использования использовал команду `awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head -10`. Здесь текстовый процессор сначала оставляет только первую строку с IP адресами, затем они сортируются в алфовитном порядке, потом удаляются повторяющиеся адреса с подсчётом повторов, потом снова сортируются, но уже по числам в обратном порядке и в конце выводятся только первые 10. Для топ-10 IP-адресов по частоте запросов в JSON-логе использовал команду `jq -r '.remote_addr' /var/log/nginx/access_json.log | sort | uniq -c | sort -rn | head -10`. Здесь утилита jq с ключом -r извлекает из каждой JSON-строки значение поля remote_addr без кавычек, оставляя только IP-адреса, затем они сортируются в алфавитном порядке, потом удаляются повторяющиеся адреса с подсчётом количества их появлений, далее результат снова сортируется, но уже по числовым значениям в обратном порядке (от большего к меньшему), и в конце выводятся только первые 10 строк, что и даёт топ-10 самых частых IP-адресов.
