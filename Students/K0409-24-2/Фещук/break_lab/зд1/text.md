Отчет по лабораторной работе №1
Описание

В данной работе была произведена обработка лог-файлов веб-сервера Nginx с использованием стандартных утилит командной строки Linux и парсера JSON-данных jq. Цель работы — извлечение статистики посещений по IP-адресам из логов разных форматов.
Этапы выполнения
1. Анализ текстового лога (access.log)

С помощью конвейера команд был выполнен сбор статистики из обычного текстового файла лога:

    Команда: awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head -10.

    Действие: Извлечение первого поля (IP-адрес), сортировка, подсчет уникальных вхождений и вывод топ-10 адресов по количеству запросов.

2. Подготовка инструментария

Для работы с логами в формате JSON была проверена доступность утилиты jq:

    Команда: sudo dnf install jq -y.

    Результат: Система подтвердила, что пакет jq-1.8.1 уже установлен и готов к работе.

3. Анализ JSON-лога (access_json.log)

Обработка структурированных данных для получения более точной статистики:

    Команда: sudo jq -r '.remote_addr' /var/log/nginx/access_json.log | sort | uniq -c | sort -rn | head -10.

    Результат: Был получен список наиболее активных IP-адресов. Лидером по количеству запросов стал адрес 10.16.59.125 (2994 запроса).
