В ходе работы в Midnight Commander (MC) были последовательно выполнены следующие действия: запуск программы командой mc, открытие верхнего меню через Esc + 9 и навигация по дереву каталогов, переключение между панелями с помощью Tab, просмотр содержимого файла через Esc + 3, создание новой папки командой Esc + 7 с именем new_folder, копирование файла file1.txt в созданную папку через Esc + 5, перемещение файла file2.txt в ту же папку с помощью Esc + 6, удаление исходного файла командой Esc + 8 и завершение работы программы через Esc + 0.
