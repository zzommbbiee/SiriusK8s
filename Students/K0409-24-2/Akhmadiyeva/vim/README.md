В ходе работы в текстовом редакторе VIM были выполнены следующие действия: сначала в пустой файл был вставлен текст с использованием режима вставки (i) и команды вставки из буфера обмена. Затем осуществлён переход в начало файла (gg) и в его конец (G), добавлена новая строка в режиме вставки (o) с последующим возвратом в нормальный режим (ESC). С помощью поиска (/Число) осуществлена навигация к строке, содержащей числовое значение, после чего применены команды инкремента (Ctrl+A) и декремента (Ctrl+X) для изменения числа. В завершение вызвана статистика файла (g Ctrl+G), произведено сохранение изменений и выход из редактора (:wq).
