# Результаты четвертой работы.
Сделал всё. Основные сложности остаются с пониманием структуры Kubernetes и ролей компонентов, но после практики стало понятнее.

## Что понял:
----------------
+ Kubernetes — система оркестрации контейнеров.
+ Кластер состоит из Control Plane и Worker-ноды.
+ Pod — минимальная единица в Kubernetes.
+ В одном Pod может быть несколько контейнеров.
+ YAML — декларативное описание объектов.
+ Kubernetes сам восстанавливает контейнеры.
