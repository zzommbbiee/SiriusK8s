# Что сделал.
Для определения топ-10 самых часто использованных IP-адресов из access.log использовал команду `awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head -10`. Сначала утилита awk извлекает из каждой строки лога первое поле, которым в стандартной конфигурации Nginx является IP-адрес. Полученный список передаётся команде sort, которая сортирует их в алфавитном порядке. Затем uniq -c группирует одинаковые адреса, идущие подряд, подсчитывает количество их повторений и выводит пары «число запросов — IP-адрес». После этого повторная сортировка sort -rn упорядочивает строки по убыванию числового значения. head -10 отсекает всё, кроме первых десяти строк. 
ДДля определения топ-10 самых часто использованных IP-адресов из JSON-файла использовал команду `jq -r '.remote_addr' /var/log/nginx/access_json.log | sort | uniq -c | sort -rn | head -10`. Утилита jq с ключом -r извлекает из каждой JSON-строки значение поля remote_addr и выводит его в виде без кавычек. Полученный список всех IP-адресов передаётся команде sort, которая сортирует их в алфавитном порядке. Затем uniq -c удаляет повторяющиеся адреса, идущие подряд, и подсчитывает количество вхождений каждого уникального IP, формируя пары «число запросов — IP-адрес». После этого выполняется повторная сортировка с помощью sort -rn. head -10 отсекает всё, кроме первых 10 адресов.
