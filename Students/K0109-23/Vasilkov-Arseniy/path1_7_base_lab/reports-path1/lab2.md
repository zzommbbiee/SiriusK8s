### Далее можно запустить с ограничениями ресурсов 
```bash
docker run -d \
  -p 5001:5000 \
  --name app-good \
  --memory="128m" \
  --cpus="0.5" \
  --restart=unless-stopped \
  myapp:good

curl localhost:5001
docker stats app-good  # и посмотреть лимиты
```
