# __лабораторная работа 1: кратко__

## что за работа
> практика по базовым механизмам linux, на которых построены контейнеры: namespaces, cgroups и chroot.

## что сделано
- проверены namespace-ы текущей системы;
- запущены изолированные окружения по pid и net;
- настроен cpu-лимит через cgroup;
- создан минимальный rootfs и выполнен chroot.

## выводы
> контейнеризация в linux строится на сочетании изоляции и лимитов ресурсов. docker только упрощает использование этих механизмов и автоматизирует рутинные шаги.
